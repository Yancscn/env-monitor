📲 添加到手机桌面
Android（Chrome）：
1. 点击浏览器右上角 ⋮ 菜单
2. 选择「添加到主屏幕」
3. 点击「添加」确认
4. 桌面出现图标，点击即可全屏打开 ✅
iPhone（Safari）：
1. 点击底部 分享 按钮 □↑
2. 下滑找到「添加到主屏幕」
3. 点击右上角「添加」
4. 桌面出现图标，点击即可全屏打开 ✅
ℹ️ 使用说明
1. 填写产品ID、设备名称、API Key 后保存
2. 📊 监控页：查看传感器数据与设备状态
3. ⚙️ 阈值页：设置5个阈值并下发到设备
4. 继电器由设备端传感器数据全自动驱动

📡 数据通过 OneNET HTTP API 轮询获取
🔒 所有配置仅存储在本地浏览器
